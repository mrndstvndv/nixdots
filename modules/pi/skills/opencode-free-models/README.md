# OpenCode Free Models Skill

This skill helps you add new free models to the OpenCode Zen provider extension.

## What This Skill Does

Documents the process for discovering and adding free AI models that work with OpenCode Zen's public API (no authentication required).

## Usage

Read the skill to understand how to:
1. Discover free models available via public API
2. Add them to `opencode-free.ts` extension
3. Verify they work correctly

The skill itself is documentation - there are no commands to run.

## Files

- `SKILL.md` - Main skill documentation
- This README - Brief overview

## Example Use Case

After completing the opencode/opencode-free extension update, this skill was created to document the process used to add:
- `nemotron-3-super-free` (1M context, 128K output)
- `mimo-v2-flash-free` (262K context, 65K output)

To opencode-free.ts alongside the existing:
- `minimax-m2.5-free` 
- `trinity-large-preview-free`
